# AI Usage Policy

Artificial Intelligence is used as a technical assistant inside a human-controlled engineering workflow.

AI may be used to:

- analyze the problem statement;
- identify constraints and missing data;
- suggest possible solution strategies;
- check reasoning consistency;
- review calculations;
- generate alternative designs;
- simulate an oral examination;
- improve technical writing.

AI is not treated as an authority.

All AI-generated content must be:

- reviewed;
- verified;
- corrected when necessary;
- integrated into a coherent human-led solution.

## Core Principle

The AI is not the author of the solution.

The AI is a tool used inside a documented reasoning and verification process.
