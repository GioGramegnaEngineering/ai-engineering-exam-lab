# Evaluation Rubric

Each case study is evaluated using the following criteria.

| Criterion | Question |
|---|---|
| Problem Understanding | Was the problem interpreted correctly? |
| Assumptions | Are the assumptions explicit and reasonable? |
| Decomposition | Was the problem broken down into manageable parts? |
| Technical Method | Is the method technically justified? |
| Calculations | Are calculations correct and traceable? |
| Verification | Were errors, limits, and edge cases checked? |
| Documentation | Is the final report clear and professional? |
| Critical Thinking | Are limitations and alternatives discussed? |
| AI Usage | Was AI used transparently and critically? |
| Defensibility | Could the solution be defended orally? |
