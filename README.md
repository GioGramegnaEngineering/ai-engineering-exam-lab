# AI Engineering Exam Lab

This repository is a public portfolio of AI-assisted engineering problem solving.

The goal is not to use AI as a shortcut, but to show how human reasoning, engineering methodology, verification, and AI-assisted workflows can be combined to solve complex technical problems.

Each case study follows a structured process:

1. problem analysis;
2. assumptions;
3. technical decomposition;
4. solution design;
5. calculations;
6. verification;
7. critical review;
8. limitations;
9. final technical report.

## Purpose

This project is designed to demonstrate:

- engineering reasoning;
- technical documentation;
- structured problem solving;
- AI literacy;
- critical validation of AI outputs;
- ability to turn exam-style problems into professional technical deliverables.
